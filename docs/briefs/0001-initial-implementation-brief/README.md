# Minegasm Next senior-developer brief

This package is a project handoff for a clean implementation of a client-side Minecraft haptics mod intended to replace the current Minegasm feature set while providing a safer, extensible multi-device architecture.

## Start here

1. `MINEGASM_NEXT_IMPLEMENTATION_BRIEF.md` - the brief (source/maintainable version).
2. `appendices/` - implementation matrices and checklists.
3. `examples/` - configuration and Buttplug message examples.
4. `assets/` - architecture diagrams (PNG + Graphviz DOT).

## Important status

- The brief is implementation-ready, but it is not compiled mod source code.
- Bootstrap the actual repository from a fresh current Stonecraft template.
- Pin all versions at kickoff; the technology landscape can change after 16 July 2026.
- Treat earlier generated MVP code bundles as disposable examples, not a foundation.
- Complete the license/name ADR before public distribution.
